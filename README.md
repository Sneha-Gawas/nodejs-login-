# NODE JS LOGIN SYSTEM
1.INSTALL NODE JS MODULES AND EXPRESS JS. EJS TEMPLATE IS USED <br>
2.INSTALL MY SQL AND CREATE DATABASE WHICH HAS TWO COLUMNS USERNAME AND PASSWORD<br>
3.CONNECT THE DATABASE BY WRITING HOSTNAME,USERNAME AND PASSWORD<br>
4.IN THIS SYSTEM,THE USER WILL REGISTER AND ITS DETAILS WILL BE SAVED IN DATABASE <br>
5.WHEN THE USER WILL ENTER USERNAME AND PASSWORD BY USING DETAILS IN DATABASE THE USER WILL BE LOGGED IN <br>
6.THE USER WILL SELECT LOGOUT
