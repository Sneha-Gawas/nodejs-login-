# NODE JS LOGIN SYSTEM
1.INSTALL NODE JS MODULES AND EXPRESS JS. EJS TEMPLATE IS USED
2.INSTALL MY SQL AND CREATE DATABASE WHICH HAS TWO COLUMNS USERNAME AND PASSWORD
3.CONNECT THE DATABASE BY WRITING HOSTNAME,USERNAME AND PASSWORD
4.IN THIS SYSTEM,THE USER WILL REGISTER AND ITS DETAILS WILL BE SAVED IN DATABASE
5.WHEN THE USER WILL ENTER USERNAME AND PASSWORD BY USING DETAILS IN DATABASE THE USER WILL BE LOGGED IN
6.THE USER WILL SELECT LOGOUT
