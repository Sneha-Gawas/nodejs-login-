# NODE JS LOGIN SYSTEM
*INSTALL NODE JS MODULES AND EXPRESS JS. EJS TEMPLATE IS USED
*INSTALL MY SQL AND CREATE DATABASE WHICH HAS TWO COLUMNS USERNAME AND PASSWORD
*CONNECT THE DATABASE BY WRITING HOSTNAME,USERNAME AND PASSWORD
*IN THIS SYSTEM,THE USER WILL REGISTER AND ITS DETAILS WILL BE SAVED IN DATABASE
*WHEN THE USER WILL ENTER USERNAME AND PASSWORD BY USING DETAILS IN DATABASE THE USER WILL BE LOGGED IN
*THE USER WILL SELECT LOGOUT
